- [ ] Why would you use class component over function components (removing hooks from the question)?

    Class components can hold state.  If you are needing to use state, before hooks, you would want to use class components

- [ ] Name three lifecycle methods and their purposes.
        render-renders the app to the DOM
        ComponentDidMount-will run the code that is in there, such as an axios call, once the component rendered in the render function
        constructor-is used to bind things to the component and also to set initial state.

- [ ] What is the purpose of a custom hook?
        Do be able to hold stateful logic in a place to be reused over and over again.
